Webhook on Request

Webhook on Request is set up using a JSON object.
The keys of the object should represent the webhook's URL, while the values should consist of a list of domain names that will activate the webhook when they are accessed.

Configuration


Example configuration

  {
    "https://mywebhook.example.com/webhook_1": [
        "shop.example.com",
        "store.example.com",
        "sales.example.com"
    ],
    "https://mywebhook.example.com/webhook_2": [
        "computer.example.com",
        "phone.example.com",
        "tablet.example.com"
    ]
  }
